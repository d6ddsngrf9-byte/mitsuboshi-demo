# 後援名義 申請の要件整理（交野市 ほか／みつぼしART星田2026）

> 目的：交野市（＋教育委員会）の「後援名義」を取得し、①信頼性の付与 ②「広報かたの」・市公式SNSへの掲載 ③施設利用料の減免可能性 につなげる。
> ※様式・締切・担当課は自治体で異なり改定もあるため、**必ず交野市公式サイト「後援名義」で最新を確認**（〔要確認〕）。まずは担当課へ電話一本が早い。

## まず確認すること（最初の一歩）
- [ ] 交野市の**担当課**を特定（文化・生涯学習・市民活動系の課が多い）。市代表→「イベントの後援名義について」で回された課がゴール。
- [ ] **申請締切**：多くの自治体で「事業開始日の1〜2か月前まで」。**会期は11/14→逆算すると9〜10月が最終ライン。今(7月)申請すれば余裕**。ただし市の様式・審査会の頻度で前後するので早期確認。
- [ ] **教育委員会後援**も取れるか（子ども・地域教育の要素があるので相性良）。別申請のことが多い。
- [ ] 後援決定「前」に、チラシ・サイトへ「交野市後援」と記載しない（決定後に解禁）。

## 一般的な提出書類（交野市の様式に合わせて差し替え）
- [ ] 後援名義使用申請書（**市の指定様式**）
- [ ] 事業計画書（目的・内容・会期・会場・対象・実施体制）
- [ ] 収支予算書（クラファン・助成金・自己資金の内訳）
- [ ] 団体概要／定款（一般社団法人 星十色）・役員名簿
- [ ] 前回（第2回）実績が分かる資料（来場・実施報告・掲載実績など）
- [ ] チラシ・ポスター案 or 企画書（未確定なら「案」でOかを確認）

## 審査で見られるポイント（通しやすくするコツ）
- **公益性**：障がいのある人もない人も参加する、地域に開かれた文化事業であること（←ここが強い）。
- **非営利性**：クラファンは開催資金の補填で、営利目的でないと明記。
- **政治・宗教に無関係**であること。
- **実現可能性**：会場（星田駅前ロータリー・星田会館等）の使用見込み、実施体制、過去2回の実績。
- **安全・保険**：会場設営や屋外展示・パレードの安全配慮、必要なら保険加入を触れておくと印象良。

## 取得後にやること（露出の最大化）
- [ ] チラシ・サイト・リリースに「交野市後援」「交野市教育委員会後援」を表記（決定通知後）。
- [ ] 「広報かたの」掲載の締切・入稿方法を担当課に確認（発行月の2か月前程度が多い）。
- [ ] 市公式SNSでの告知可否・タイミングを相談。
- [ ] 施設（星田会館 等）の**利用料減免**が後援に紐づくか確認。

## スケジュール目安（会期 2026/11/14〜29 逆算）
| 時期 | やること |
|---|---|
| **7〜8月（今）** | 担当課特定・様式入手・締切確認 → 申請書一式を準備・提出 |
| 9月 | 審査・後援決定 → 表記解禁、広報かたの入稿 |
| 10月 | 市SNS告知調整、施設減免手続き |
| 11月 | 会期・広報かたの掲載号で露出 |

- 大阪府（文化・福祉系）／北河内の近隣市の後援は要件が重め・締切早めなので、取れそうなら早期に。
